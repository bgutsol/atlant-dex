## How to build

Install dependencies
``` bash
npm install
```
Serve with hot reload at localhost:8080
``` bash
npm run dev
```
Build for production with minification
``` bash
npm run build
```

## Recommended nodejs and npm version
nodejs version >= 8.8.1 <br />
npm version >= 5.4.2
